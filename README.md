# jprojets

Voilà le service complet avec **TOUTES** les méthodes nécessaires ! 

**Résumé de ce qui est inclus :**

✅ **Gestion Utilisateur** : connexion, déconnexion, vérification
✅ **CRUD Projets complet** : création, lecture, mise à jour, suppression
✅ **Gestion des images projets** : ajout, suppression (par index ou path), suppression totale
✅ **Gestion des tâches** : CRUD complet, toggle completion
✅ **Gestion des sous-tâches** : CRUD complet, toggle completion
✅ **Gestion des checklists** : ajout, mise à jour, suppression
✅ **Gestion des statuts** : changement et filtrage
✅ **CRUD Informations complet** : création, lecture, mise à jour, suppression
✅ **Gestion des points d'information** : ajout, mise à jour, suppression
✅ **Gestion des images informations** : ajout, suppression (par index ou path), suppression totale
✅ **Recherche hyperpuissante projets** : 13 options de filtrage et tri
✅ **Recherche hyperpuissante informations** : 8 options de filtrage et tri
✅ **KPIs Projets** : 6 statistiques différentes
✅ **KPIs Informations** : statistiques jour/semaine/mois/total
✅ **Utilitaires** : fermeture, réinitialisation, export de données
